CloudBees mobile services

This project showcases CloudBees mobile services. We have assembled a showcase of several template projects that you can refer to, build on that show the full range of CloudBees mobile services, from building mobile apps, deploying server apps, integrating data, push messages - all hosted, by us.

The demo app

You really wanted to build your own restaurant review mobile app right? No? Oh, well lucky this is just a demo, it is called "Gasp" which is a pun on "Yelp".

This app is made up of iOS and Android apps, as well as integration, databases, server apps, and more. Restaurants are reviewed and geotagged, updates to restaurant views are pushed out to phones once a change in the data is detected.
