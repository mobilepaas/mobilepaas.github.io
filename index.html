CloudBees mobile services

This project showcases CloudBees mobile services. We have assembled a showcase of several template projects that you can refer to, build on that show the full range of CloudBees mobile services, from building mobile apps, deploying server apps, integrating data, push messages - all hosted, by us.

The demo app

This app is made up of iOS and Android apps, as well as integration, databases, server apps, and more - all to build a "restaurant review" service - called "gasp" - A pun on "yelp" - get it? Nevermind, it is just a sample. Restaurants are reviewed and geotagged, updates to restaurant views are pushed out to phones once a change in the data is detected.
